Install magento:
git clone https://github.com/weblips/magento2.git .
composer install
Install with CLI:
php bin/magento setup:install --base-url="http://yoururl.com/" --db-host="localhost" --db-name="dbname" --db-user="dbuser" --db-password="dbpass" --admin-firstname="admin" --admin-lastname="admin" --admin-email="user@example.com" --admin-user="admin" --admin-password="admin123" --language="en_US" --currency="USD" --timezone="America/Chicago" --use-rewrites="1" --backend-frontname="admin"
Err:
The requested PHP extension ext-intl * is missing from your system. Install or enable PHP's intl extension.
in php.ini uncomment php_intl or download this lib

Simple module:

1. Create folder : app/code/Weblips/Test

2. Registration module:

etc/module.xml

                    
<?xml version="1.0"?>
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:Module/etc/module.xsd"> 
<module name="Weblips_Test" setup_version="0.0.1"/> 
</config>
  • name="Weblips_Test" - your`s Namespase of module
  • setup_version="0.0.1" - yor`s version for update and links Setup

3. Create you registration:

registration.php

                    
<?php 
\Magento\Framework\Component\ComponentRegistrar::register( 
    \Magento\Framework\Component\ComponentRegistrar::MODULE, 
    'Weblips_Test', 
    __DIR__ 
);
  • 'Weblips_Test' - is you module name in systems (and yor`s namespace)

4. (IF YOU NEED) composer.json for update module

composer.json

                    
{
    "name": "weblips/test",
    "description": "N/A",
    "require": {
        "php": "~5.6.5|7.0.2|7.0.4|~7.0.6",
        "magento/module-catalog-rule": "100.2.*", // зависимости
        "magento/module-sample-data": "100.2.*"
    },
    "type": "magento2-module", // указывает что это модуль
    "version": "100.2.0-dev", // версия согласно установки из etc/module.xml
    "license": [
        "OSL-3.0",
        "AFL-3.0"
    ],
    "autoload": {
        "files": [
            "registration.php"
        ],
        "psr-4": {
            "Weblips\\Test\\": "" //namespace
        }
    }
}

5. Create routing:

etc/frontend/routes.xml // for frontend/routes

etc/adminhtml/routes.xml // for backend

                    
<?xml version="1.0"?> 
<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="../../../../../../lib/internal/Magento/Framework/App/etc/routes.xsd"> 
    <router id="standard"> 
        <route id="example" frontName="example"> 
            <module name="Weblips_Test" /> 
        </route> 
    </router> 
</config>
  • router id="standard" - default router
  • route id="example" frontName="example" // unicum http://site.com/example - for frontend/routes
